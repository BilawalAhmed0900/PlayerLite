# PlayerLite
A small audio player for geeks

# Requirement
* `.Net Framework 2.0`

# Compilation
* Clone this repo `git clone https://github.com/Dragneel1234/PlayerLite.git`
* Open  `PlayerLite.sln` in Visual Studio
* Build `Release`, simple as that
